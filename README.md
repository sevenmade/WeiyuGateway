# Weiyu Gateway (Unofficial) for Home Assistant

> 非官方 Home Assistant 自定义集成，用于本地直连网关与断路器设备。  
> 本项目仅用于学习、研究与个人自用分享，不提供任何官方背书或商业承诺。

![Weiyu Logo](./custom_components/weiyu_gateway/logo.png)

## 项目简介

本项目用于在 Home Assistant 中本地直连网关，监控并控制智能断路器/漏电保护断路器，无需中转云服务。

当前已支持：

- UDP 自动发现网关
- 子设备开关控制（单设备）
- 网关总开关（一键全部合闸/分闸）
- 遥测与状态（电压、电流、功率、电能、运行状态等）
- 告警相关实体
- 漏电自检按钮（针对支持设备）
- 网关参数实体与一键下发全部设备

## 安装

### 方式一：手动安装

将 `custom_components/weiyu_gateway` 复制到 Home Assistant 的 `/config/custom_components/`，然后重启 Home Assistant。

### 方式二：HACS 自定义仓库

仓库已包含 `hacs.json`，可作为自定义仓库导入 HACS。

## 使用说明（简要）

1. 在 HA 中添加集成：`Weiyu Gateway`
2. 推荐先用“自动发现”选择网关
3. 确保 `listen_ip` 为 HA 所在局域网地址，`listen_port` 为未占用端口
4. 添加成功后在设备页可见网关与子设备实体
