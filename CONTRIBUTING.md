# 参与贡献

感谢你愿意改进本集成。

## Issue

- 尽量说明：Home Assistant 版本、集成版本（`manifest.json` 中 `version`）、网关型号/固件、网络拓扑（HA 与网关是否同网段）
- 附上相关日志片段（可打码 IP），以及期望行为 vs 实际行为

## Pull Request

- 保持单次 PR **范围小、目的单一**，便于审阅与回归
- 与设备协议相关的改动，如能附上一段**匿名化**的报文或文档出处，会很有帮助
- 提交信息建议使用英文或中英文简短说明（例如 `fix: ...` / `feat: ...`）

## 代码风格

- 与现有文件保持一致的类型标注、日志与异步写法
- 避免无关格式化或大范围重排

## 法律与商标

- 请勿在仓库中提交受版权限制且未获授权的厂商完整文档、固件或密钥材料
- 若需引用公开协议摘要，请控制在合理引用范围内，并注明来源
